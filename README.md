# Utada Hikaru Intro

一个为宇多田光制作的艺术化个人页面，包含成长时间线、从童年到成熟期的创作阶段，以及中文、英文、日文三语切换。

Live:

- [utada-hikaru-intro.vercel.app](https://utada-hikaru-intro.vercel.app)

## Features

- Artistic landing page with editorial layout
- Timeline of life and career milestones
- Songwriting eras from childhood to the present
- Chinese, English, and Japanese language switcher
- Pure static implementation for easy deployment

## Files

- `index.html`: page structure
- `styles.css`: visual design and responsive layout
- `script.js`: multilingual content and rendering logic

## Local Preview

If you want to preview it locally, any static file server will work.
